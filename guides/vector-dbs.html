Vector Databases

Pinecone, Weaviate, ChromaDB — storing and searching embeddings.

What Are Vector DBs?

They store embeddings and enable fast similarity search using metrics like cosine or dot product.

Popular Tools

  • FAISS: Facebook AI Similarity Search — fast and flexible.
  • Weaviate: Schema-aware, supports hybrid search.
  • ChromaDB: Lightweight, open-source, great for RAG pipelines.

Indexing Strategies

Use flat, IVF, HNSW, or PQ depending on speed vs accuracy tradeoffs.
